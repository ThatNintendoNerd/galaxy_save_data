# galaxy_save_core_derive

![crates.io](https://img.shields.io/crates/v/galaxy_save_core_derive)
![docs.rs](https://img.shields.io/docsrs/galaxy_save_core_derive)

galaxy_save_core_derive is a Rust library for defining derive macros for common traits used in the save data of Super Mario Galaxy and Super Mario Galaxy 2.
