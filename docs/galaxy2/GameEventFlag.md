# Game Event Flags

A Game Event Flag is a key-value pair for a Boolean.

The default value for all flags is `false`.

## ハチマリオ初変身

> Bee Mario First Transformation

Determines if the player was informed about collecting a Bee Mushroom for the first time.

## テレサマリオ初変身

## アイスマリオ初変身

> Ice Mario First Transformation

Determines if the player was informed about collecting an Ice Flower for the first time.

> [!NOTE]
> This flag goes unused in the vanilla game but can still be set if the Ice Flower is inserted into a stage and collecting it for the first time.

> World 1 First Play

Determines if World 1 was introduced to the player.

This flag is used to start the sequence upon traveling to World 1 for the first time.

## 最終ギャラクシー出現

> Final Galaxy Appearance

Determines if the Grandmaster Galaxy is available.

> [!NOTE]
> If this flag is manually cleared after collecting one of its Stars, the galaxy will remain available but the path to it will have closed off.

## ノーマルエンディングメール送信

> Normal Ending Mail Sent

Determines if the message following the player's partial completion of the game was sent to the Wii Message Board.

> [!NOTE]
> This flag is only set if the player has accumulated less than 120 Stars after collecting the Grand Star from Bowser's Galaxy Generator and both [`ノーマルエンディング実行`](#ノーマルエンディング実行) and [`スター120個エンディング実行`](#スター120個エンディング実行) are cleared.

## コンプリートメール送信

> Complete Mail Sent

Determines if the message following the player's full completion of the game was sent to the Wii Message Board.

## スターピースカウンターストップ

> Star Piece Counter Stop

Determines if 9,999 Star Bits were collected.

This flag is used to swap coconuts with watermelons.

## コインカウンターストップ

> Coin Counter Stop

Determines if 9,999 coins were collected.

This flag is used to enable ghosts for all galaxies.

## ルイージプレイ済

> Luigi Played

Determines if Luigi was played as.

## でしゃばりルイージ出現中

> Intrusive Luigi Appearing

Determines if Luigi can appear on standby in a mission he has not appeared in before.

This flag is set after Luigi first talks to Mario on Starship Mario and when [`でしゃばりルイージ出現カウンタ`](https://github.com/ThatNintendoNerd/galaxy_save_data/blob/main/docs/galaxy2/GameEventValue.md#でしゃばりルイージ出現カウンタ) is subsequently a multiple of six.

## ゲームオーバーで終了

> Game Over and then Exit

Determines if the player got a game over, then saved and quit when prompted.

This flag is used to suggest the player visit another galaxy as if they had accumulated a multiple of three game overs.

> [!NOTE]
> The effects of this flag are clobbered after being tested by the game. This is presumably an artifact of altered behavior from Super Mario Galaxy which prompted the player to save and quit after a game over. If the player manually sets this flag, its effects can be properly observed.
