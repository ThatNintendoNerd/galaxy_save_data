# Game Event Flags

A Game Event Flag is a key-value pair for a Boolean.

The default value for all flags is `false`.

## ハチマリオ初変身

> Bee Mario First Transformation

Determines if the player was informed about collecting a Bee Mushroom for the first time.

## テレサマリオ初変身

> Teresa Mario First Transformation

## LuigiTalkAfterRescued

Determines if Luigi was talked to since he was rescued from the Ghostly Galaxy.

This flag is used to prepare his hide-and-seek event.

## RosettaTalkAboutTico

Determines if Rosalina was talked to in the second mission of the Gateway Galaxy.

This flag is used to prevent repeated interactions.

## RosettaTalkAfterNormalEnding

Determines if Rosalina was talked to after the normal ending in the Comet Observatory.

This flag is used to prevent repeated interactions.

## ViewCompleteEnding

Determines if the 120-Star ending was achieved.

This flag is used on the File Selection screen to display the second crown and enable the ability to switch between Mario and Luigi.

## StarPieceCounterStop

Determines if 9,999 Star Bits were collected.

This flag is used to swap coconuts with watermelons.
